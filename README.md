# НПП «ГКС» - Сайт компании

Современный сайт компании НПП «ГКС», разработанный на Vite + React.

## Технологии

- **Vite** - быстрый сборщик проектов
- **React** - библиотека для создания пользовательских интерфейсов
- **React Router** - маршрутизация
- **CSS** - стилизация

## Установка

```bash
npm install
```

## Запуск проекта

Для запуска проекта в режиме разработки:

```bash
npm run dev
```

Для сборки проекта:

```bash
npm run build
```

Для предпросмотра собранного проекта:

```bash
npm run preview
```

## Структура проекта

```
src/
├── components/          # Переиспользуемые компоненты
│   ├── home/           # Компоненты главной страницы
│   ├── Header.jsx      # Шапка сайта
│   └── Footer.jsx      # Подвал сайта
├── pages/              # Страницы сайта
│   ├── Home.jsx        # Главная страница
│   ├── About.jsx       # О компании
│   ├── Products.jsx    # Продукция
│   ├── Services.jsx    # Услуги
│   ├── News.jsx        # Новости
│   ├── Career.jsx      # Карьера
│   └── Contacts.jsx    # Контакты
├── styles/             # Глобальные стили
└── App.jsx             # Главный компонент приложения
```

## Страницы сайта

- **Главная** (`/`) - главная страница с основной информацией
- **О компании** (`/about`) - информация о компании
- **Продукция** (`/products`) - каталог продукции
- **Услуги** (`/services`) - перечень услуг
- **Новости** (`/news`) - новости компании
- **Карьера** (`/career`) - вакансии и карьерные возможности
- **Контакты** (`/contacts`) - контактная информация
